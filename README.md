# MERN Stack Quiz App

This project is a **full-stack Quiz Application** built using the **PERN** stack (PostgreSQL, Express, React, Node.js). The app allows users to take quizzes and admins to create quizzes and manage users. It features multiple admin roles, user dashboards, and results.

## Features

- **Create Quiz**: Admin users can create new quizzes with multiple questions.
- **Multiple Admins**: Allows different admin roles to manage the quiz app.
- **User Dashboard**: Users have a personalized dashboard to track their quiz results and participation.
- **Take Quiz**: Users can take quizzes, receive scores, and view correct answers after submission.
- **Quiz Results**: Users and admins can view results, including correct answers and score breakdown.

## Tech Stack

- **Frontend**: 
  - **React.js**: For building the user interface with state management using hooks.
  
- **Backend**:
  - **Node.js**: A JavaScript runtime used to build the backend.
  - **Express.js**: Web application framework for handling HTTP requests.
  - **PostgreSQL**: SQL database for storing quiz questions, results, and user data.
  

- **Styling**:
  - **Tailwind CSS**: Tailwind CSS for UI components.
